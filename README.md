# eslint-config-px
vue3版本的eslint团队规范


## Installation

Install @memory13/eslint-config-px:

```
npm install --save-dev @memory13/eslint-config-px
```

```javascript
// eslinttrc.js
{
  "extends": [
    "@memory13/eslint-config-px"
  ]
}
```
```javascript
// package.json 
{
  "eslintConfig": {
    "extends": "@memory13/eslint-config-px"
  }
}
```
